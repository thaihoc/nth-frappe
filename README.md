* MariaDB v10.11
* Redis v7 (Cache, Queue, và SockerIO)
* Frappe ERPNext & HRMS v15

Cài đặt trên Podman: [Podman installation](docs/podman-installation.md)

Cài đặt trên K8s: [Kubernetes installation](docs/k8s-installation.md)
